Qui sommes-nous

artisanAI est un outil d'assistance administrative destiné aux artisans et TPE français. Il vous aide à générer rapidement vos devis, factures et relances.

Quelles informations sont utilisées

Les informations que vous saisissez — nom de votre entreprise, coordonnées, numéros de documents — ainsi que les descriptions que vous tapez ou dictez pour créer vos documents.

Où sont enregistrées vos données

Toutes vos informations (profil entreprise, compteurs de numérotation) sont enregistrées uniquement sur votre téléphone ou ordinateur. Aucune donnée n'est envoyée sur un serveur ni conservée par artisanAI.

Ce qui est transmis pour générer vos documents

Quand vous décrivez un document, votre message est transmis de façon sécurisée à un service d'intelligence artificielle pour générer le devis ou la facture. Ce message n'est pas conservé ni réutilisé une fois le document créé.

Vos droits

Vous pouvez à tout moment supprimer l'ensemble de vos données en cliquant sur "Supprimer mes données" dans la section Mon entreprise. Cette action effacera toutes les informations enregistrées sur votre appareil.

Nous contacter

Pour toute question relative à vos données personnelles, contactez-nous à : contact@artisanai.fr (adresse en cours d'activation)
